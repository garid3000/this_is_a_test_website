1. What

You can host simple static website (something like a blog) on your github. (or host a documentation webpages of your project).

2. Why

2.1. Advantages:

  • easy (apparently quite easy)
  • free (as beer)

2.2. Disadvantages:

  • not free (as freedom)
  • kinda "spoon feeds" you

3. How

3.1. Step 1: creation of repo

  • create empty repo on github
  • clone it into local machine

3.2. Step 2: creation of html

  • create a index.html in the root directory
  • commit & push to back to github

3.3. Step 3: configuring github pages

  1. Go to Settings inside you repo
  2. Go to Pages on the left panel
  3. Select Deploy from branch
  4. (optional) customize other domain you have
